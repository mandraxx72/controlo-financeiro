# Controlo Financeiro

Aplicação de controlo financeiro pessoal desenvolvida em HTML, CSS e JavaScript.

## Funcionalidades

- ✅ Dashboard com resumo mensal
- ✅ Gestão de transações (receitas/despesas)
- ✅ Categorias personalizáveis
- ✅ Metas de orçamento
- ✅ Múltiplas carteiras
- ✅ Transferências entre contas
- ✅ Conversor CVE/EUR
- ✅ Calendário financeiro
- ✅ Exportação CSV e Backup JSON
- ✅ PWA (instalável no telemóvel)
- ✅ Modo escuro/claro

## Como Usar

1. Acesse: https://SEU_USUARIO.github.io/controlo-financeiro
2. No telemóvel, adicione ao ecrã inicial para usar como app

## Tecnologias

- HTML5
- CSS3
- JavaScript (ES6+)
- Chart.js
- LocalStorage
